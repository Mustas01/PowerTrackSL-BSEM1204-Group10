About PowerTrack SL

Empowering citizens through utility metrics transparency and responsive grid metrics feedback loops.

Our Core Architecture Goals

PowerTrack SL was conceptualized to solve structural information bottlenecks between community members and power distribution controllers in Sierra Leone. By tracking outage patterns and crowdsourcing local infrastructure alerts, we create a transparent and actionable map of power stability across the country.

Academic Affiliation

This web workspace represents a formal practical design submission for the Web Design 1 Module at Limkokwing University. The design highlights modern CSS properties, responsive grids, and clean UI components built to solve real-world utility issues.
