About PowerTrack SL

Transforming electricity monitoring in Sierra Leone through real-time data, community reporting, and actionable energy insights.

Our Mission

Energy infrastructure remains one of the foundational cornerstones of socioeconomic development within Sierra Leone. However, a persistent challenge for domestic households and commercial enterprises alike has been the lack of real-time, centralized data regarding power distribution states and maintenance schedules. PowerTrack SL was conceptualized to solve this critical information asymmetric bottleneck. By engineering a dual-purpose data pipeline, the platform transforms ordinary citizens from passive consumers into active node reporters.

How It Works

Our architecture blends official grid operational inputs with structured, crowdsourced public fault reporting. The result is an intuitive, live dashboard interface that maps out grid stability, displays immediate appliance energy analytics, and delivers localized telemetry to keep the public safely informed.

Core Framework

  • Responsive Design: Semantic layout scaling for all device sizes
  • Accessibility: Adaptive typography and high contrast for readability
  • Security: Safe front-end patterns and data handling

Academic Project

Engineered to satisfy high-level UX requirements for the Web Design 1 Module framework at Limkokwing University. This project demonstrates modern web design principles including responsive layouts, semantic HTML, and user-centered design thinking.
