# PowerTrack SL ⚡

## Project Overview

PowerTrack SL is a web-based electricity outage monitoring and reporting platform designed to address electricity-related challenges in Sierra Leone.

The platform allows users to:

* View power outage updates
* Report electricity problems
* Access emergency contacts
* Learn energy-saving tips

This project is developed for the Web Design 1 module at Limkokwing University.
